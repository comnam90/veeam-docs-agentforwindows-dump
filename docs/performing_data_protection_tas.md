# Performing Data Protection Tasks


You can perform the following data protection tasks:

* Back up your data and store the resulting backup files in one of the following types of Veeam backup repositories:

* [In a backup repository managed by a Veeam backup server](integration_backup_repository.md)
* [In a Veeam Cloud Connect repository](integration_backup_cloud.md)
* [Copy Veeam Agent backups from the backup repository to a secondary backup repository with backup copy jobs](integration_backup_copy.md).
* [Use SureBackup](sn_surebackup.md).
* [Archive Veeam Agent backups to tapes with backup to tape jobs](integration_tape.md).
* [Scan Veeam Agent backups](integration_backup_scan.md).
