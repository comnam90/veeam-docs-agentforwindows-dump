# Assigning Roles to Users


User roles configured on the Veeam backup server apply to Veeam Agent backup jobs as well.

To learn more, see the [Users and Roles](https://helpcenter.veeam.com/docs/vbr/userguide/users_roles.html?ver=13) section in the Veeam Backup & Replication User Guide.
