# Creating Veeam Recovery Media with UI


To create a Veeam Recovery Media with the UI, do the following:

1. [Launch the Create Recovery Media wizard](image_create_launch.md).
2. [Specify recovery media options](image_create_options.md).
3. [Specify a path to the ISO](image_create_path.md).
4. [Review recovery image settings](image_create_review.md).
5. [Take additional steps to finalize recovery media creation](image_create_after.md).
