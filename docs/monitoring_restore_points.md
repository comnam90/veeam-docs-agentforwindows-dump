# Viewing Statistics for Separate Restore Points


You can view the following information about separate restore points in the backup chain:

* [View general statistics](monitoring_restore_points_general.md) — for any separate restore point.
* [View transaction log backup statistics](monitoring_restore_points_db.md) — for restore points created by the backup job with transaction log backup enabled.
