# Restoring from Veeam Recovery Media in the Manual Mode


When you perform restore from the Veeam Recovery Media in the manual mode, you must manually specify backup file location and restore point. Take the steps below to recover your computer:

1. [Boot from the Veeam Recovery Media](baremetal_boot.md).
2. [Select the network adapter or wireless network](baremetal_network.md).
3. [Launch the Veeam Recovery Media wizard](baremetal_launch.md).
4. [Specify the backup file location](baremetal_source.md).
5. [Select the remote storage type](baremetal_remote.md).
6. [Specify remote storage settings](baremetal_remote_conf.md).
7. [Select a backup](baremetal_backup.md).
8. [Select a restore point](baremetal_point.md).
9. [Select the data restore mode](baremetal_mode.md).
10. [Map restored disks](baremetal_disk_mapping.md).
11. [Resize restored volumes](baremetal_resize.md).
12. [Start the restore process](baremetal_start.md).
