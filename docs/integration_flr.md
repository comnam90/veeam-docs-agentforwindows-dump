# Restoring Files and Folders


You can use the Veeam Backup & Replication console to restore individual files and folders from Veeam Agent backups.

Before you perform file-level restore, [check prerequisites](integration_file_restore_before.md). Then use the File Level Restore wizard to restore the necessary files and folders.

1. [Launch the File Level Restore wizard](integration_file_restore_launch.md).
2. [Select a machine](integration_file_restore_machine.md).
3. [Select a restore point](integration_file_restore_point.md).
4. [Specify a restore reason](integration_file_restore_reason.md).
5. [Verify restore settings](integration_file_restore_settings.md).
6. [Finalize restore](integration_file_restore_complete.md).
