# Step 4. Specify Remote Storage Settings


Specify settings for the remote storage that contains a backup file from which you plan to restore data:

* [Object storage settings](files_restore_object_storage.md) — if you have selected the Object storage option at the [Remote Storage](files_restore_remote.md) step of the wizard.
* [Shared folder settings](files_restore_share.md) — if you have selected the Shared folder option at the [Remote Storage](files_restore_remote.md) step of the wizard.
* [Veeam backup repository settings](files_restore_vbr.md) — if you have selected the Veeam backup repository option at the [Remote Storage](files_restore_remote.md) step of the wizard.
* [Veeam Cloud Connect repository settings](files_restore_sp.md) — if you have selected the Veeam Cloud Connect repository option at the [Remote Storage](files_restore_remote.md) step of the wizard.
