# Step 4. Specify Backup Scope Settings


Specify backup scope for the backup job:

* [Select volumes to back up](backup_job_volumes.md) — if you have selected the Volume level backup option at the [Backup Mode](backup_job_mode.md) step of the wizard.
* [Select folders to back up](backup_job_folders.md) — if you have selected the File level backup option at the [Backup Mode](backup_job_mode.md) step of the wizard.
