# Reporting


Veeam Agent for Microsoft Windows provides several ways to get information about performed backups:

* [You can view information about performed backups in the Control Panel](reporting_control_panel.md).
* [You can get information about the backup state using the Veeam Agent for Microsoft Windows tray agent](reporting_tray.md).
* [You can get information about the backup progress using the Veeam Agent for Microsoft Windows taskbar button](reporting_taskbar.md).
* [You can get information about Veeam Agent for Microsoft Windows events using the events bar in the Control Panel](viewing_events.md).
* [You can get information about Veeam Agent for Microsoft Windows events using Windows Notification Center](viewing_events_desktop.md).
* [You can get information about performed backups in email reports](reporting_email.md).
